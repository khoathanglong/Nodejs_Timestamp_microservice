API Basejump: Timestamp microservice

User Story

  • I can pass a string as a parameter, and it will check to see whether that string contains either a unix timestamp or a natural language date (example: Sat Jan 17 1970).
  • If it does, it returns both the Unix timestamp and the natural language form of that date.
  • If it does not contain a date or Unix timestamp, it returns null for those properties.
  • Example Usage
    • /December%2015,%202015
    • /1450137600
  • Example output: { "unix": 1450137600, "natural": "Sat Jan 17 1970" }
